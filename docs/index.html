GitHub Actions를 통해 예약 가능한 주차 시간대를 자동으로 조회합니다.

⚙️ GitHub 설정

토큰은 브라우저 로컬스토리지에만 저장됩니다. 토큰 발급하기 → (scopes: workflow, contents)

🔍 조회 조건
