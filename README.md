# Automation

A collection of automation projects.

## Projects

### airport-parking-checker

인천공항 예약 주차 사이트에서 특정 날짜의 예약 가능한 입출차 시간 조합을 조회하는 프로그램입니다.

**Tech Stack:** Node.js, TypeScript

**Usage:**
```bash
cd airport-parking-checker
pnpm install
pnpm run dev
```

자세한 내용은 [airport-parking-checker/README.md](./airport-parking-checker/README.md)를 참조하세요.
